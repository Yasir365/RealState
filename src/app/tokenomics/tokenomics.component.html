Token Supply and Distribution

Maximum Token Supply

REAL has a fixed maximum supply of 100 million tokens, creating exclusivity over time.

Token Allocation

  • Founders and Team: 4% allocated for long-term development.
  • Start-Up: 2% for technology growth.
  • Liquidity Pool Burn: 2% burned for security.
  • Community Engagement: 92% for property acquisition.

How the Deflationary Model Works

  • Rental Income and Token Buybacks: Profits from REAL properties buy REAL tokens from the market.
  • Token Burn Process: Purchased tokens are permanently removed, reducing total supply.
  • Increased Exclusivity: Each token represents a larger portion of resources and benefits.

REAL’s Passive Income Strategy

  • Sustained Income: Continuous rental property income provides steady resources.
  • Funding Token Burns: Income supports ongoing buybacks and token burns.
  • Membership Benefits: Exclusivity increases as token burns continue, enhancing user experience.
